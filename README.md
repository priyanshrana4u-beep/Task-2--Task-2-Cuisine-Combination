# Task-2-Cuisine-Combination

## Project Overview
This project analyzes restaurant data from the Zomato dataset using Python. The project explores cuisine combinations, cleans restaurant ratings, and identifies the highest-rated cuisine combinations through data visualization.

## Objectives
- Explore the Zomato dataset
- Identify the most common cuisine combinations
- Analyze average ratings for different cuisine combinations
- Visualize the results using bar charts

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
- zomato.csv

## Project Workflow
1. Import the dataset.
2. Explore the dataset using `head()` and `info()`.
3. Check for missing values.
4. Clean the `rate` column.
5. Find the top 10 most common cuisine combinations.
6. Calculate the average rating for each cuisine combination.
7. Visualize the results using bar charts.

## Results
- Identified the 10 most common cuisine combinations.
- Identified the 10 highest-rated cuisine combinations.
- Created visualizations to compare cuisine popularity and ratings.
